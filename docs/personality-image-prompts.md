# SKTI 人格形象图 Prompts

32 个人格形象的生成 prompt。

## 使用说明

- **风格锚点**：以第一张 SOFA（`public/images/personalities/sofa.png`）为风格参考图喂给模型，保持整套视觉一致。
- **默认 prompt**（已在生成器里预设，无需重复写）：扁平矢量插画风格，几何切块，柔和色块阴影，风格化 2D，透明背景。
- **每个 prompt 只描述角色本身**（姿势、表情、服装、道具、雪板类型），不含背景和风格关键词。
- **雪板类型**：单板 / 双板已标注在每个人格后面。双板 = 两根板 + 两根雪杖；单板 = 一块板。
- **命名规则**：生成后按 slug 保存到 `public/images/personalities/<slug>.png`。
