# SKTI - Ski Type Indicator 设计文档

## 产品定位

SKTI（Ski Type Indicator）是一个面向中国滑雪圈的娱乐社交向人格测试网站。用户通过回答约 30 道题目，测出自己属于哪种「滑雪人格」。风格对标 SBTI（sbti.dev），抽象搞笑、有梗、适合社交传播。

- 目标用户：中国滑雪爱好者
- 语言：中文为主，后续可扩展
- 核心卖点：搞笑人格 + 滑雪场景共鸣 + 分享传播

## 维度体系

6 组维度，混合严肃与搞笑：

| # | 维度名 | 两极 | 类型 |
|---|--------|------|------|
| S1 | 技术风格 | 稳健派 ↔ 莽夫派 | 严肃 |
| S2 | 选道偏好 | 道内安全员 ↔ 野雪探险家 | 严肃 |
| S3 | 社交属性 | 独狼 ↔ 雪场社牛 | 混搭 |
| S4 | 装扮执念 | 随便套个羽绒服 ↔ 雪场时装周 | 搞笑 |
| S5 | 摸鱼指数 | 滑到缆车停 ↔ 第三趟就去吃火锅 | 搞笑 |
| S6 | 装备心态 | 能用就行 ↔ 全套顶配还没学会刹车 | 搞笑 |

S1-S2 是核心滑雪能力维度，S3 是社交属性，S4-S6 是搞笑/梗维度。每个维度是连续谱，最终根据 6 个维度的组合匹配人格类型。

## 题目设计

- 约 30 道题，每题 3 个选项（A/B/C），B 通常是中间态
- 题目随机排序，严肃题和搞笑题穿插出现
- 每道题对应 1-2 个维度

### 题目风格示例

严肃型：
- 「缆车上你一般在干嘛？」A. 复盘刚才那条线 / B. 刷手机 / C. 跟旁边陌生人聊今天雪况

搞笑/无厘头型：
- 「你摔倒的时候雪镜飞了，帽子也飞了，手套也飞了，你第一反应？」A. 先找手机看碎没碎 / B. 保持摔倒的姿势不动，假装在做雪天使 / C. 大喊"有没有人帮我录到了"
- 「如果你的滑雪板会说话，它最可能说的是？」A. "求你了轻点" / B. "我们又来这条道了？第47次了" / C. "你根本配不上我"
- 「你在雪场看到有人穿西装打领带在滑雪，你？」A. 这人比我有排面 / B. 过去问他是不是刚开完会 / C. 明天我也穿
- 「你在缆车上，前面的人雪板上绑了一只毛绒企鹅，你？」A. 我也想要一只 / B. 默默拍照发朋友圈"今日雪场显眼包" / C. 下缆车后跟着他滑了三趟试图搭讪问哪买的

## 人格类型

约 25 种人格，每种包含：代号（英文）、中文名、slogan、人格描述（约200字，搞笑但有洞察）、AI 生成形象图。

### 示例人格

| 代号 | 中文名 | Slogan | 特征 |
|------|--------|--------|------|
| SOFA | 沙发客 | "我来雪场是为了那碗火锅" | 摸鱼指数拉满，装扮精致，滑两趟就去休息室，朋友圈全是雪场美食 |
| YOLO | 莽王 | "这条道没死过人吧？那我试试" | 技术激进，什么道都冲，摔了也觉得帅，装备全是摔出来的战损版 |
| GUCCI | 雪场名媛 | "雪服比雪技重要一万倍" | 装扮执念顶级，装备全套顶配，拍照时间比滑雪时间长，但确实好看 |
| GPS | 领航员 | "跟我走，这条道我闭眼都能滑" | 技术稳健，热衷带新人，对雪场地图倒背如流，社交属性强 |
| 404 | 失联者 | "别找我，我在树林里" | 独狼型野雪爱好者，手机没信号，下午三点还没人知道他在哪 |
| BLING | 装备党 | "板还没开封，但先发个开箱视频" | 装备顶配但技术入门，雪具室最亮的仔 |
| NPC | 背景板 | "我就在初级道来回，你们玩你们的" | 佛系到极致，技术一般也不想进步，但每周都来，享受过程 |
| DRAMA | 戏精 | "啊——我的膝盖——算了没事继续滑" | 摔倒反应夸张，社交属性爆棚，全雪场都认识他 |
| ICE | 冷面杀手 | "少说话，多刷道" | 技术强，独狼，从开门滑到关门，不社交不拍照不吃饭 |
| WIFI | 信号猎人 | "这个位置有4G吗" | 缆车上发抖音，雪道上直播，休息时修图，滑雪是内容创作的副产品 |

完整的 25 种人格将在实现阶段补充，覆盖 6 个维度的各种组合。

## 页面结构

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 产品介绍、开始测试入口、人格类型预览 |
| 测试页 | `/test` | 30道题逐题作答，进度条，支持上一题 |
| 结果页 | `/result/[type]?s=xxx` | 人格结果展示，`s` 参数编码维度得分 |
| 人格总览 | `/types` | 所有25种人格的卡片浏览 |
| 人格详情 | `/type/[slug]` | 单个人格的完整介绍（不带个人得分） |
| 关于 | `/about` | 测试说明、维度介绍 |

## 用户流程

```
首页 → 点击"开始测试" → 逐题作答（30题，A/B/C）→ 答完自动计算 → 跳转结果页
```

- 题目随机排序（可重新随机）
- 进度条显示当前题号/总题数
- 支持"上一题"回退修改
- 答完即跳转，无需额外确认

## 结果页

1. **人格卡片**：代号 + 中文名 + AI 生成形象图 + slogan
2. **人格描述**：约200字搞笑但有洞察的描述
3. **维度落点图**：6 个维度的条形图，显示用户在每个维度的位置
4. **分享功能**：复制链接、下载分享卡（前端生成图片）
5. **推荐人格**：「看看其他几种滑雪人格」

## URL 参数编码

结果通过 URL 参数编码，分享链接即可还原结果页，不需要后端。

- URL 示例：`/result/sofa?s=2a1b3c2b1a3c`
- 6 个维度得分（0-100）各用 2 位 base36 字符表示
- 总共 12 个字符
- 不带 `s` 参数访问 `/result/sofa` 时显示该人格的通用介绍

## 技术架构

### 技术栈

- **框架**：Next.js 14（App Router），静态导出（`output: 'export'`）
- **样式**：Tailwind CSS
- **动画**：Framer Motion（页面切换、题目切换、结果揭晓）
- **分享卡生成**：html2canvas
- **部署**：Vercel

### 计分逻辑

1. 每道题对应 1-2 个维度，选 A/B/C 分别给该维度 +1/+2/+3 分
2. 30 题答完后，6 个维度各有一个总分
3. 每个维度归一化到 0-100
4. 用 6 个维度的高/中/低组合，匹配最接近的人格类型
5. 将维度得分编码进 URL 参数

### 视觉风格

- 类似 SBTI：干净简约、浅色调
- 主色调：冷色系（浅蓝/冰白），呼应滑雪主题
- 每种人格配 AI 生成的卡通形象（先占位，后续可替换）
- Framer Motion 实现题目切换动画和结果揭晓效果
